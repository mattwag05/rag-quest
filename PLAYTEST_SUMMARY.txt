   - Status: OPEN

2. CombatEncounter Constructor (rag-quest-m0h)
   - Constructor: (player_character, enemies: List[Enemy])
   - Docs show: (player=..., enemy=..., location=...)
   - Status: OPEN

3. SaveManager API Completely Redesigned (rag-quest-pxd)

   - Docs: save_game(world_name, slot, state, character_name)

   - Docs: list_saves(world_name)

   - Docs: load_game(world_name, slot)
   - Status: OPEN

   - Status: OPEN
