# BUG-001: Inventory.list_items() Returns Wrong Type

**Severity:** P1
**Status:** Open

list_items() returns a list instead of dict[str, Item]. Causes AttributeError when code tries to access dict methods.

**Location:** rag_quest/engine/inventory.py

**Reproduction:**
```python
inv = Inventory()
inv.add_item("Sword", "A blade", quantity=1, weight=5)
items = inv.list_items()
# Returns: ['common_item', 'uncommon_item', ...] instead of dict
```

**Expected Behavior:** Return dict[str, Item] mapping names to items
